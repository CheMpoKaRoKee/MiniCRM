# Test log

## Что сделал

Собрал рабочий прототип мини-CRM для юридической команды:

- Web App для добавления клиентов;
- Google Sheets как база данных;
- хранение контактных данных клиента;
- юридические статусы;
- комментарии по делу;
- счетчики по статусам;
- email-уведомление юристу;
- логирование действий.

## Что делал сам

Самостоятельно определил сценарий, структуру таблиц, набор юридических статусов, UX-логику и стек. Проверял, чтобы прототип был понятен не как техническая заготовка, а как инструмент для юриста или менеджера.

## Как использовал AI

AI использовался как ускоритель разработки: помогал генерировать код Google Apps Script и HTML-интерфейса, находить ошибки, улучшать UX, готовить README и структурировать итоговую подачу.

## Почему Google Sheets + Apps Script

Для MVP за короткое время этот стек дает быстрый результат без сервера, отдельного хостинга, базы данных и платной инфраструктуры. Google Sheets позволяет прозрачно проверить данные, а Apps Script позволяет сделать Web App, backend-логику и email-уведомления.

## Бонус

Добавлена автоматизация email-уведомления юристу при добавлении нового клиента. Email задается на листе `Settings`, результат отправки фиксируется в колонке `notification_status`.
